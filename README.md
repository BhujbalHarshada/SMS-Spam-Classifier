# SMS Spam Classifier

A Machine Learning based SMS Spam Detection web application built using Python, Scikit-learn and Flask.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Flask
- HTML
- CSS

## Features

- SMS Spam Detection
- Text Preprocessing
- TF-IDF Vectorization
- Naive Bayes Classifier
- Logistic Regression Comparison
- Flask Web Application
